# Changelog

## 2.0.0 (2026-05-11)

- **--full**: Save entire page as markdown (nav, footer, CTAs included)
- **--extract**: Extract emails, phones, addresses, social links from all pages
- **--output**: Custom output directory path
- **--max-pages**: Configurable page limit
- Bundle includes crawl date metadata
- JavaScript, CSS, JSON, XML files excluded from crawling
- Extracted data saved as `_extracted.json`
- GitHub Actions release workflow
